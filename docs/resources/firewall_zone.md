# unifi_firewall_zone (Resource)

Manages a UniFi firewall zone. Supports creating custom zones and managing network assignments for both user-defined and system-defined zones.

## Example Usage

### Create a custom zone

```terraform
resource "unifi_firewall_zone" "iot" {
  name        = "IoT"
  network_ids = [data.unifi_network.iot_network.id]
}
```

### Import an existing zone by name and manage its networks

```terraform
# First, import the zone:
#   terraform import unifi_firewall_zone.servers "Servers"
#
# Or by UUID:
#   terraform import unifi_firewall_zone.servers "dfb21062-8ea0-4dca-b1d8-1eb3da00e58b"

resource "unifi_firewall_zone" "servers" {
  name        = "Servers"
  network_ids = [
    data.unifi_network.server_vlan.id,
    data.unifi_network.management.id,
  ]
}
```

### Use with firewall policies

```terraform
resource "unifi_firewall_zone" "iot" {
  name        = "IoT"
  network_ids = [data.unifi_network.iot.id]
}

resource "unifi_fw" "block_iot_to_lan" {
  name    = "block-iot-to-lan"
  enabled = true

  action {
    type = "DROP"
  }

  source {
    zone_id = unifi_firewall_zone.iot.id
  }

  destination {
    zone_id = data.unifi_firewall_zone.default.id
  }

  ip_protocol_scope {
    ip_version = "IPV4_AND_IPV6"
  }
}
```

## Import

Zones can be imported by UUID or by name:

```shell
# By UUID
terraform import unifi_firewall_zone.example "dfb21062-8ea0-4dca-b1d8-1eb3da00e58b"

# By name (case-insensitive)
terraform import unifi_firewall_zone.example "My Zone"
```
